# GameJam
* #### 장르 : 탑뷰 모바일 슈팅 게임
* #### 개발 언어 : C# / Unity3d
* #### 디자인 툴 : 

## 프로젝트 소개
### 게임 소개
* 스마일게이트 챌린지 중 게임잼에서 제작된 탑뷰 모바일 슈팅 게임 프로토타입입니다.
* 좌측 조이스틱으로 이동이 가능하며, 우측 버튼으로 공격을 차지할 수 있습니다.
* 데미지를 입힐 시 조이스틱 옆 게이지가 차게 되며, 게이지가 모두찰 시 궁극기를 사용할 수 있습니다.
* 4인 멀티 게임으로 구상하고 있으나, 현 프로토타입에서는 AI와 대전하는 것으로 구현되었습니다.
